# vue3-app

vue3 app

## Framework packages

Vite + Vue + Pinia + Vue Router + TypeScript + TDesign Vue Next

## Package Manager

pnpm

## Quick start

```
git clone https://github.com/xinlei3166/vite-vue-template.git
```

## Project setup

```
pnpm install
```

### Compiles and hot-reloads for development

```
pnpm run dev
```

### Compiles and minifies for production

```
pnpm run build
```

### Run your tests

```
pnpm run test
```

### Lints and fixes files

```
pnpm run lint
```
